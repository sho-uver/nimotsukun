# 遊び方
この[exe](https://github.com/sho-uver/nimotsukun/releaseslatest/download/nimotsukun.exe)をダウンロードして実行するとゲームが起動します。
ルールと操作方法を確認して遊んでください。

# ルール
- プレイヤー(p)を移動させて荷物(o)を押して運び、すべての荷物をゴール(.)の上に置いたらクリア
- プレイヤーの移動先が壁である場合は移動できない
- 荷物を押した先に荷物か壁がある場合も移動できない

## 補足ルール
- p,oはゴールの上に移動すると大文字のP,Oに変わる
- 荷物は押し出すのみで引くことや横にスライドさせることはできない

# 操作方法
- wキー:プレイヤーが上に移動する
- zキー:プレイヤーが下に移動する
- aキー:プレイヤーが左に移動する
- sキー:プレイヤーが右に移動する
- qキー:ゲーム終了

# 実装で考えたこと
- mainには必要最低限の処理だけ記載し、将来的にメニュー画面などを実装してもstateパターンで切り替えて処理を進行できるようにした。
- GameManagerにゲームの進行役を持たせた。
- Stageにステージデータを所持させて情報の受け渡しをさせるようにした。プレイヤーの移動等によるデータの変更はGameManagerに責任を持たせた。
- ゲームオブジェクトのクラスをそれぞれ作ることで将来他の機能を追加しやすいようにした。(例えばプレイヤーが攻撃するとか)
- コマンドパターンとして入力した値をInputCommandに変換することで、w,a,s,zキー以外の入力が増えても対応できるようにした。
